## 项目概况

Chrome MV3 扩展，功能：上传图片 → VL/OCR 识别文字 → 商品区域检测过滤 → 等效字数计数。技术栈 Vite + TypeScript + React + Zustand + ONNX Runtime WASM。构建走 esbuild IIFE，dist/ 入库。测试 vitest 5 文件 53 用例全绿（2026-06-29 实测）。
